1. Скачиваем tar.xz архив для нужной архитектуры отсюда: https://desktop.telegram.org/

2. Распаковываем в домашнюю директорию, например:

mkdir ~/.apps

cd ~/Downloads

tar xpvf tsetup.*.tar.xz -C ~/.apps/

3. Запускаем Telegram:

~/.apps/Telegram/Telegram

4. Пользуемся. Приложение будет спокойно обновляться до новых версий по мере выхода.

В Ubuntu можно закрепить значок приложения на панели запуска.
