Чтобы настроить расстояние между маркером и текстом в списке, например, вместо:


* Some list text goes

here.


Сделать:


* Some list text goes

here.


Или:


*Some list text goes

here.


Такое можно сделать следующим образом:


li span {

position: relative;

left: -10px;

}


<ul>

<li><span>item 1</span></li>

<li><span>item 2</span></li>

<li><span>item 3</span></li>

</ul>


Пространство регулируется при помощи свойства left.
