1. Скачать файл репозитория отсюда. Нужно выбирать версию 9.6, а не 10 и не 11. В новых версиях репозитория нет pgAdmin. Качать отсюда: https://yum.postgresql.org/


2. Установить скачанный файл в систему:

sudo dnf install ./pgdg-fedora96-9.6-4.noarch.rpm

3. Найти название пакета pgAdmin 4:

sudo dnf search pgadmin

4. Установить пакет pgAdmin 4:

sudo dnf install pgadmin4-v2
