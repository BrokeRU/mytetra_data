1. Скачать Postman:


curl -L https://dl.pstmn.io/download/latest/linux64 -o postman-linux-x64.tar.gz


2. Распаковать в директорию ~/.apps/


tar xvzf postman-linux-x64.tar.gz -C ~/.apps


3. Создать символическую ссылку для запуска:


sudo ln -s ~/.apps/Postman/Postman /usr/local/bin/postman


4. Создать .desktop-файл:


cat <<EOT >> ~/.local/share/applications/postman.desktop

[Desktop Entry]
Name=Postman
GenericName=API Client
X-GNOME-FullName=Postman API Client
Comment=Make and view REST API calls and responses
Keywords=api;
Exec=postman
Terminal=false
Type=Application
Icon=/home/br0ke/.apps/Postman/resources/app/assets/icon.png
Categories=Development;Utilities;

EOT


5. Обновить список программ:


kbuildsycoca5
