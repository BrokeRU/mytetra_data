1. Скачать свежую версию программы отсюда: https://webhamster.ru/site/page/index/articles/projectcode/105

2. Распаковать её, например, в папку .apps:

tar xf mytetra_1_43_25_lin_64.tar.gz -C ~/.apps/


3. Создать .desktop-файл следующей командой:

cat <<EOT >> $HOME/.local/share/applications/mytetra.desktop

[Desktop Entry]

Name=MyTetra

Comment=Smart manager for information collecting

Comment[ru]=Менеджер накопления осмысленной информации

Exec=env LD_LIBRARY_PATH=$HOME/.apps/mytetra_1_43_25_lin_64/lib /home/br0ke/.apps/mytetra_1_43_25_lin_64/mytetra

Icon=$HOME/.apps/mytetra_1_43_25_lin_64/mytetra.png

Terminal=false

Type=Application

StartupNotify=true

Categories=Qt;Utility;

EOT


4. Запустить MyTetra по созданному ярлыку.

5. Настроить доступ к git-репозиторию по ssh-ключам.

6. Настроить синхронизацию MyTetra:

git config --global user.email "and-semakin@ya.ru"
git config --global user.name "Andrey Semakin"
cd ~/.config/mytetra/data/
rm -rf base/ icons/ database.ini mytetra.xml
git clone git@github.com:BrokeRU/mytetra_data.git .

7. Запустить MyTetra, зайти в настройки синхронизации и вставить в поле ввода следующую команду, установить галочки напротив "синхронизировать при запуске", "синхронизировать при выходе", "периодические синхронизировать":

cd %a ; git add . ; git commit -a -m MyTetraCommit ; git pull -s recursive ; git push


8. Перезапустить MyTetra и пользоваться.
