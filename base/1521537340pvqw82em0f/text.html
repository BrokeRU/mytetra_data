1. Скачать rpm отсюда: https://copr.fedorainfracloud.org/coprs/ollowtf/mytetra/package/mytetra/

2. Установить на Ubuntu alien:

sudo apt-get install alien dpkg-dev debhelper build-essential

3. Сконвертировать rpm в deb:

sudo alien mytetra-1.42.2.-1.x86_64.rpm

4. Установить deb:

sudo dpkg -i mytetra-1.42.2.-1.x86_64.deb
