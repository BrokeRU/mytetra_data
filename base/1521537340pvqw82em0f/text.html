1. Скачать rpm для Fedora отсюда: https://copr.fedorainfracloud.org/coprs/ollowtf/mytetra/package/mytetra/

2. Установить на Ubuntu alien:


sudo apt-get install alien dpkg-dev debhelper build-essential


3. Сконвертировать rpm в deb:


sudo alien mytetra-1.42.2.-1.x86_64.rpm


4. Установить deb:


sudo dpkg -i mytetra-1.42.2.-1.x86_64.deb


5. Установить Qt v5.7.1 (см. соответствующую заметку)

6. Отредактировать .desktop-файл MyTetra, в секции Exec добавить перед именем исполняемого файла путь к библиотекам Qt:


...

Exec=env LD_LIBRARY_PATH=/home/br0ke/Qt5.7.1/5.7/gcc_64/lib

...


В общем это должно выглядеть так:


[Desktop Entry]

Name=MyTetra

Comment=Smart manager for information collecting

Comment[ru]=Менеджер накопления осмысленной информации

Exec=env LD_LIBRARY_PATH=/home/br0ke/Qt5.7.1/5.7/gcc_64/lib mytetra

Icon=mytetra

Terminal=false

Type=Application

StartupNotify=true

Categories=Qt;Utility;


7. Перезагрузить компьютер или перелогиниться.

8. Можно запускать MyTetra из меню Ubuntu.


Настроить интеграцию с git в MyTetra:

9. Нужно произвести первичную настройку Git -- задать имя и e-mail.

10. Нужно настроить доступ к репозиторию по ключу ssh.

11. Узнать, где располагается директория с данными MyTetra:

11.1. Запускаем MyTetra;

11.2. Заходим в "Инструменты" -> "Настройки" -> "Основные";

11.3. Запоминаем путь в поле "Директория с данными";

11.4. Дальнейшие операции выполняем из этой директории;

12. Очищаем папку с данными полностью:


rm -rf ./*


13. Клонируем в текущую директорию удаленный репозиторий:


git clone git@github.com:BrokeRU/mytetra_data.git


14. В настройках синхронизации MyTetra указать команду синхронизации:


cd %a ; git add . ; git commit -a -m MyTetraCommit ; git pull -s recursive ; git push


14. Запускаем программу и можем пользоваться.
