Для установки необходимо иметь аккаунт на qt.io либо придется создать его в процессе установки. Установка подходит только для 64-битных ОС.


1. Скачиваем установщик (примерно 700 МБ):

Для версии 5.7.1:

wget http://download.qt.io/official_releases/qt/5.7/5.7.1/qt-opensource-linux-x64-5.7.1.run

Для версии 5.6.2:

wget http://download.qt.io/official_releases/qt/5.6/5.6.2/qt-opensource-linux-x64-5.6.2.run


2. Добавляем скачанному файлу права на исполнение:

chmod +x qt-opensource-linux-x64-*.run


3. Запускаем скачанный файл на исполнение:

./qt-opensource-linux-x64-*.run


4. Производим установку в графическом режиме: выбираем путь установки, устанавливаемые компоненты. В процессе установки нужно войти в Qt Account.
