В Ubuntu есть чудесный инструмент, который позволяет просто установить все инструменты, которые нужны разработчику на Ubuntu. Называется Ubuntu-make или просто umake.


Установить umake:


sudo add-apt-repository ppa:ubuntu-desktop/ubuntu-make

sudo apt-get update

sudo apt-get install ubuntu-make


Посмотреть список доступных инструментов:

umake --help


Установить выбранный инструмент, например, IDE PyCharm:

sudo umake ide pycharm


Удалить PyCharm:

umake -r ide pycharm
