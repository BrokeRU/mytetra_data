1. Скачать официальный образ:

docker pull telegrammessenger/proxy:latest


2. Создать папку:

mkdir proxy-config


3. Запустить:

docker run -d -p443:443 --name=mtproto-proxy --restart=always -v proxy-config:/data telegrammessenger/proxy:latest


4. Скопировать ссылку для подключения:

docker logs -f mtproto-proxy


5. Подключиться, используя клиент Telegram;


6. Мониторинг:

docker exec mtproto-proxy curl http://localhost:2398/stats


Количество подключенных клиентов:

docker exec mtproto-proxy curl http://localhost:2398/stats | grep total_special_connections
