Vue-cli -- консольная утилита, которая позволяет создавать шаблоны для новых проектов на основе чистого html, webpack или browserify. В своей работе использует Node. Так же утилита умеет запускать сервер разработчика с горячей перезагрузкой страницы при изменениях в коде.


Установка:

sudo npm install -g vue-cli

или

sudo npm install -g @vue/cli


Создание нового проекта на основе шаблона webpack:

vue init webpack my-project


Список доступных шаблонов:

  • simple -- простой шаблон html с подключенным Vue из CDN;
  • webpack-simple -- шаблон на основе Webpack + vue-loader;
  • webpack -- шаблон на основе Webpack + vue-loader + обработка CSS + тестирование + линтинг;
  • pwa -- шаблон на основе Progressive Web Applications;
  • browserify-simple -- простой шаблон на основе Browserify;
  • browserify -- продвинутый шаблон на основе Browserify.
