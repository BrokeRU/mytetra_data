Получить текущий конфиг кластера:


kubectl config view



Получить список пространств имен:


kubectl get namespaces



Получить список подов:


kubectl get pods



Получить список подов во всех пространствах имен:


kubectl get pods --all-namespaces



Получить список репликасетов:


kubectl get replicasets



Получить список деплойментов:


kubectl get deployments



Получить список сервисов:


kubectl get services



Получить список ингрессов:


kubectl get ingress
