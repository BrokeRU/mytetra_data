1. Скачать свежий ESXi Host Client здесь в формате *.vib: https://labs.vmware.com/flings/esxi-embedded-host-client#summary


2. Загрузить скачанный файл на ESXi-хост (допустим, в корень):

scp esxui-signed-8122819.vib root@192.168.99.3:/


3. Перевести хост в Maintainance Mode.


4. Установить пакет:

esxcli software vib install -v esxui-signed-8122819.vib

5. Вывести хост из Maintainance Mode.


6. Пользоваться без надоедливых ошибок.
