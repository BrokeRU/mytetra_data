1. Сделать бэкап важных данных.


2. Обновить зависимости системы:


sudo dnf upgrade --refresh


3. Перезагрузить машину.


4. Установить dnf-plugin-system-upgrade:


sudo dnf install dnf-plugin-system-upgrade


5. Скачать пакеты для обновления (заменить 30 на новую версию):


sudo dnf system-upgrade download --refresh --releasever=30


6. Если возникли какие-то проблемы с зависимостями (такое часто бывает со сторонними пакетами), то можно повторить предыдущую команду с флагом --allowerasing (нужно внимательно просмотреть список удаляемых пакетов и подтвердить). Так же можно добавить флаг --best, если проблема с неразрешенными зависимостями.


7. Установить следующий релиз, система уйдет в перезагрузку:


sudo dnf system-upgrade reboot


8. Дождаться завершения процесса (примерно 1 час).
