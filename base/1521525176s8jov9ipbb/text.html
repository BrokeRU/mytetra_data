Meld -- графическая утилита, позволяющая просматривать разницу (diff) между файлами, выполнять слияния (merge). Удобно при использовании с системами контроля версий (Git, SVN).

Написана на Python.


Установка на Ubuntu:

sudo apt-get install meld


Установка на Fedora:

sudo dnf install meld


Сборка последней версии из исходников:

sudo apt-get install intltool itstool gir1.2-gtksource-3.0 libxml2-utils

git clone https://git.gnome.org/browse/meld

cd meld

sudo python setup.py install

После этого Meld должен появиться в меню Приложений.
