1. Установить eslint-plugin-import:


npm install --save-dev eslint-plugin-import


2. Прописать в .eslintrc:


{

"settings": {

"import/resolver": {

"node": {

"paths": ["src"]

}

}

}

}



Если разрешение путей уже настроено в webpack, то можно использовать конфигурацию из wepkack:


3. Установить eslint-import-resolver-webpack:


npm install --save-dev eslint-import-resolver-webpack


4. Прописать в .eslint.json:


"settings": {

"import/resolver": {

"webpack": {

"config": "webpack.prod.js"

}

}

}
