1. Добавить репозитории QGIS в систему:


sudo sh -c 'echo "deb http://qgis.org/debian $(lsb_release -sc) main" >> /etc/apt/sources.list'

sudo sh -c 'echo "deb-src http://qgis.org/debian $(lsb_release -sc)" >> /etc/apt/sources.list'

wget -O - http://qgis.org/downloads/qgis-2017.gpg.key | gpg --import

gpg --fingerprint CAEB3DC3BDF7FB45

gpg --export --armor CAEB3DC3BDF7FB45 | sudo apt-key add -


2. Установить QGIS:


sudo apt-get update && sudo apt-get install qgis python-qgis
