1. Перейти по ссылке и авторизоваться:

http://www.vmware.com/go/download-vmrc


2. Скачать bundle-файл по ссылке "VMware Remote Console for Linux".


3. Добавить права на исполнение скачанному файлу:

chmod +x VMware-Remote-Console-10.0.2-7096020.x86_64.bundle


4. Запустить файл с sudo:

sudo ./VMware-Remote-Console-10.0.2-7096020.x86_64.bundle


5. Пройти графический мастер установки.


6. Удалить:


Показать список установленных программ VMware:

vmware-installer -l


Удалить VMware Remote Console:

vmware-installer --uninstall-product vmware-vmrc
