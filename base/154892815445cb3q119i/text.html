process -- глобальная переменная, которая позволяет получить или изменить сведения о выполняемом процессе.


1. Получить значение переменной окружения DEBUG:


console.log(process.env.DEBUG);


Переменные окружения -- всегда строки!


2. Получить значение переменной окружения или значение по умолчанию:


const DEBUG = process.env.DEBUG || false;

const PORT = process.env.PORT || 3000;


3. Установить переменную окружения:


process.env.MY_VARIABLE = 'ahoy';
