1. Числовые типы:


  1. Integer -- целые числа произвольной длины
  2. Int -- целые числа фиксированной длины
  3. Float -- числа с плавающей точкой
  4. Double -- числа с плавающей точкой двойной точности


2. Булевый тип:

True

False


3. Символы и строки:

Символы записываются в одинарных кавычках и могут содержать произвольный символ Unicode.

'a'

'Й'

'\n'


Строки -- это списки символов, но также имеют собственный синтаксис. Строковые литералы можно записывать в двойных кавычках:


"Hello world"
