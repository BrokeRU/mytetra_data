Бывает нужно при создании нового репозитория создать пустую ветку master, а всю работу проделать в другой ветке, а затем создать Merge Request, чтобы другие участники команды могли поревьювить код.


1. Создать ветку без родителей:


git checkout --orphan <branchname>


2. Очистить содержимое рабочей директории:


git rm --cached -r .

git reset --hard
