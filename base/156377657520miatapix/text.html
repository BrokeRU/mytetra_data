Cargo -- это система сборки и пакетный менеджер для Rust.


1. Проверить установлен ли Cargo:


cargo --version


2. Создать новый проект при помощи Cargo:


cargo new hello_cargo

cd hello_cargo


Эта команда создаст директорию проекта с несколькими файлами внутри, а также инициализирует Git.


3. Собрать проект при помощи Cargo и выполнить собранную программу:


cargo build

./target/debug/hello_cargo


4. То же самое в одну команду:


cargo run


5. Проверить, компилируется ли программа:


cargo check


6. Собрать программу с оптимизациями для релиза:


cargo build --release
