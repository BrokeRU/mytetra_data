Чтобы выполнить рекурсивный поиск по файлам в текущей директории (и поддиректориях), можно использовать команду grep:


grep -r "texthere" .


Можно использовать следующие полезные ключи:

  • -R -- переходить по символьным ссылкам;
  • -i -- поиск без учета регистра;
  • -n -- добавить номера строк в вывод.
