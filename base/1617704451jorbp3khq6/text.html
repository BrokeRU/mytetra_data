1. Открыть панель команд: Ctrl+Shift+P (Cmd+Shift+P);

2. Набрать settings.json, чтобы открыть файл конфигурации;

3. Настроить рулеры, как удобно.


Одна штука:


"editor.rulers": [79]


Много:


"editor.rulers": [80,120]


Для разных типов файлов:


"[git-commit]": {"editor.rulers": [50]},

"[python]": {

"editor.rulers": [

79,

120

]

}


С выбором цветов:


"editor.rulers": [

{

"column": 80,

"color": "#ff00FF"

},

100, // <- a ruler in the default color or as customized (with "editorRuler.foreground") at column 100

{

"column": 120,

"color": "#ff0000"

},

],
