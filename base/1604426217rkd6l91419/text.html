Есть несколько способов вывести данные в stdout.


1. fmt.Println()


Всегда добавляет пробелы между аргументами. В конце добавляет символ переноса строки.


2. fmt.Print()


Добавляет пробелы между аргументами, только если среди них нет строк. Не добавляет символ переноса строки.



Примеры про пробелы:


fmt.Print("Ivan", 27) // Ivan27 потому что есть строка


fmt.Println("Ivan", 27) // Ivan 27 потому что Println всегда добавляет пробелы


fmt.Print(33, 27) // 33 27 потому что нет строк
