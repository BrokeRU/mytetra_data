Установка:


Ctrl + P:


ext install fatihacet.gitlab-workflow


Настройка:


1. Создать токен доступа в аккаунт на GitLab: https://gitlab.com/profile/personal_access_tokens

2. Ctrl + Shift + P:


GitLab: Set GitLab Personal Access Token


Возможности:

  • Просмотр статуса пайплайна, запуск/перезапуск пайплайна;
  • Просмотр списка MR, issues;
  • Редактирование и валидация .gitlab-ci.yml;
  • Создание сниппетов;
  • Открыть текущий файл и строку на GitLab.
