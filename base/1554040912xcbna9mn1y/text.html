В спецификации ECMAScript 2017 у строк появился новый метод padStart():


str.padStart(targetLength [, padString])


Примеры:


'abc'.padStart(10); // " abc"

'abc'.padStart(10, "foo"); // "foofoofabc"

'abc'.padStart(6,"123465"); // "123abc"

'abc'.padStart(8, "0"); // "00000abc"

'abc'.padStart(1); // "abc"
