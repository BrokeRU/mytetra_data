Чтение данных из стандартного потока ввода производится при помощи fmt.Scan(&a).


var name string

fmt.Print("Введите имя: ")

fmt.Scan(&name)


Точно так же можно считывать и числа:


var age int

fmt.Print("Введите возраст: ")

fmt.Scan(&age)


Можно считать сразу несколько переменных:


fmt.Scan(&a, &b, &c)


Если вводятся строки, то ввод будет обрезаться после первого пробела.
