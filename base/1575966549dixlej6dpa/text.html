По умолчанию терминал MacOS настроен так, что при завершении шелла или скрипта окно терминала остается открытым с надписью:


[Process completed]


 Чтобы этого избежать, и просто закрывать терминал по завершению работающего в нем процесса, нужно зайти в настройки Terminal, выбрать активный профиль (должен быть помечен default), перейти на вкладку Shell и в пункте "When the shell exits" выбрать "Close if the shell exited cleanly". Теперь терминал можно закрывать при помощи Ctrl+D.
