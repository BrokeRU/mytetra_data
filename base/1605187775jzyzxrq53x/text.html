Срез -- это гомогенная (данные должны быть одного типа) последовательность переменной длинны. В других языках подобные структуры данных называют динамическими массивами или списками.


Массивы и срезы тесно связаны. Срез -- это структура данных, строится поверх массива, и предоставляет доступ к элементами базового массива.


Срез состоит из следующих трёх компонентов:

ссылка на первый доступный через срез элемент базового массива

длина (length) -- количество элементов в срезе
