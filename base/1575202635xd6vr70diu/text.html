Чтобы узнать на какие коммиты указывают тэги в Git можно выполнить следующую команду:


git show-ref --tags


Результат:


f727215 refs/tags/v2.16.0

56072ac refs/tags/v2.17.0

b670805 refs/tags/v2.17.1

250ed01 refs/tags/v2.17.2


Если известно конкретное название тэга, то можно так:


git show <tag>


или


git log -1 <tag>
