1. Установить Docker:


brew cask install docker


2. Запустить Docker:


  • Нажать Cmd+Space и набрать Docker.
  • В окне Docker needs privileged access нажать ОК
  • Ввести пароль и нажать ОК


Когда Docker запускается, он создает символические ссылки на команды docker, docker-compose, docker-machine и docker-credential-osxkeychain в /usr/local/bin.


В системном трее должен появится логотип Docker. Можно кликнуть по нему, чтобы увидеть статус системы (starting/running).


3. Проверить, что все установилось:


docker run --rm hello-world


4. Установить автодополнения:


brew install bash-completion

brew install docker-completion

brew install docker-compose-completion

brew install docker-machine-completion
