По умолчанию, Ubuntu называет папки в домашней каталоге пользователя именами из системного языка. На русском языку получаются следующие названия:

  • Рабочий стол;
  • Загрузки;
  • Документы;
  • Изображения.


Зачастую их неудобно набирать в терминале, потому что приходится переключать раскладку клавиатуры.


Чтобы переименовать эти директории в англоязычные аналоги, можно использовать следующий способ:

1. Переключить системный язык на английский (Параметры системы > Язык системы > переместить English в начало списка > Применить для всей системы).

2. Перелогиниться или перезагрузить машину.

3. При входе в систему выскочит сообщение, предлагающее переименовать папки в домашнем каталоге пользователя в соответствии с новым языком. Соглашаемся.

4. Меняем язык обратно на русский.

5. Перелогиниваемся.


Чтобы произвольным образом переименовать эти папки, нужно:

1. Создать новые директории.

2. Отредактировать соответствующим образом файл ~/.config/user-dirs.dirs .

3. Выполнить команду xdg-user-dirs-update.
