Vue.js (читается как "вью") -- прогрессивный реактивный фреймворк. Я бы описал его как AngularJS с элементами React. Vue действительно очень похож на Angular своими директивами, а от React он заимствует декларативное описание элементов интерфейса. По заверениям разработчиков всё должно работать очень быстро и эффективно. Самый популярный JS-фреймворк на начало 2018 года.


Домашняя страница фреймворка: https://vuejs.org/


Простейший скрипт:


HTML:

<div id="app">
{{ message }}
</div>


JS:

var app = new Vue({
el: '#app',
data: {
message: 'Hello Vue!'
}
})


Результат:

Hello Vue!
