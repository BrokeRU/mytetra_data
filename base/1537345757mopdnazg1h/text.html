1. Найти имя нужного пода:


kubectl get po


2. Выполнить команду внутри пода:


kubectl exec -it my-pod -- /bin/bash


3. Если под содержит больше одного контейнера:


kubectl exec -it my-pod --container main-app -- /bin/bash


4. Если команда не подразумевает интерактивности, то можно убрать ключи -it:


kubectl exec my-podcat /etc/nginx/conf.d/default.conf
