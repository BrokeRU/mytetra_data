Допустим, мы подключаем ноутбук к внешнему монитору, внешний монитор настраиваем основным экраном, и всё работает. Но форма для ввода пароля при логине всё равно будет показываться на экране ноутбука. Потому что это настраивается отдельно.


1. Проверяем, что тип сессии у текущего пользователя и пользователя gdm совпадает:


grep -r -e Session /var/lib/AccountsService/users


2. Скопируем текущий конфиг экранов пользователю gdm:


sudo cp -f ~/.config/monitors.xml ~gdm/.config/monitors.xml

sudo chown $(id -u gdm):$(id -g gdm) ~gdm/.config/monitors.xml

sudo restorecon ~gdm/.config/monitors.xml
