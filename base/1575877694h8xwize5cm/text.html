Чтобы получить версию файла базы данных (2 или 3), можно посмотреть Magic Header String. Для версии 2 это первые 48 байтов, для версии 3 -- первые 16 байтов.


$ head -c 48 file2.db

** This file contains an SQLite 2.1 database **


$ head -c 16 file3.db

SQLite format 3


Более простой способ:


$ file file2.db

file2.db: SQLite 2.x database


$ file file3.db

file3.db: SQLite 3.x database
