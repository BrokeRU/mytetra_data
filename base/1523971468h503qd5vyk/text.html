1. Скачать GitKraken:


wget https://release.gitkraken.com/linux/gitkraken-amd64.tar.gz


2. Переместить в нужную директорию, распаковать и удалить архив:


mv gitkraken-amd64.tar.gz ~/.apps

tar -xvzf gitkraken-amd64.tar.gz
rm gitkraken-amd64.tar.gz


3. Добавить в переменную $PATH:


echo "export PATH=\$PATH:~/.apps/gitkraken" >> ~/.bashrc

source ~/.bashrc


4. Скачать картинку, положить в нужное место:


Скачать отсюда:

https://drive.google.com/file/d/0B-3KQ_ohu-RFVkJyS1Zfa2NLSVE/view


cp ~/Загрузки/422255.png ~/.apps/gitkraken/gitkraken.png


5. Создать desktop-файл:


cat <<EOT >> ~/.local/share/applications/gitkraken.desktop

[Desktop Entry]
Name=GitKraken
Comment=Git Flow
Exec=/home/br0ke/.apps/gitkraken/gitkraken
Icon=/home/br0ke/.apps/gitkraken/gitkraken.png
Terminal=false
Type=Application
Encoding=UTF-8
Categories=Utility;Development;

EOT


6. Доустановить нужные библиотеки:


sudo dnf install libXScrnSaver libcurl-7.55.1-10.fc27.x86_64
sudo ln -s /usr/lib64/libcurl.so.4 /usr/lib64/libcurl-gnutls.so.4


7. Обновить меню:


kbuildsycoca5


Если программа не запускается, то нужно запустить ее из терминала и посмотреть ошибку.
