К сожалению, в Chrome нет хоткея для выхода из адресной строки, поэтому если начать редактировать, то после этого уже никак без помощи мыши не вернуться на страницу. Это особенно бесит, если пользоваться такими плагинами как Vimium.


Воркэраунд:

  1. Перейти в Settings;
  2. Перейти в Search Engine;
  3. Выбрать Manage search engines;
  4. Нажать Add;
  5. Заполнить форму следующим образом:
    • Search engine: exit address <- любое понятное название
    • Keyword: q <- клавиша или слово, по которому можно будет выйти из адресной строки
    • URL: javascript: <- именно так (с двоеточием на конце)
  6. Нажать Add.


Теперь при попадании в адресную строку оттуда можно выйти, нажав q, Enter.
