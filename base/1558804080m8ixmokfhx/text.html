Jest добавляет в глобальную область видимости следующие переменные (список не исчерпывающий). Их не нужно ниоткуда импортировать.


  • expect -- функция проверки, аналог assert в pytest;
  • test -- функция для создания тест-кейса, который содержит внутри себя один или несколько вызовов expect; так же имеет синоним it;
  • describe -- функция, используется для группировки тест-кейсов; может быть вложена в себя сколько угодно раз.


describe('myFunc', () => {

it('should return 1', () => {

expect(myFunc()).toBe(1);

});

});
