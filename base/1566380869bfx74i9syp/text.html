1. Установить asdf (см. соответствующую заметку).

2. Установить плагин asdf-nodejs:


asdf plugin-add nodejs https://github.com/asdf-vm/asdf-nodejs.git


3. Импортировать ключи, которыми подписывают релизные бинарники Node, чтобы при установке они верифицировались автоматически (обязательный шаг):


bash ~/.asdf/plugins/nodejs/bin/import-release-team-keyring


4. Установить выбранные версии Node, например:


asdf install nodejs 10.14.2

asdf install nodejs 12.8.1


5. Использовать:


При выборе любой версии может потребоваться выбор глобальной версии (если имеется глобально установленный Node), можно указать в качестве глобальной версии системную:


asdf global nodejs system


Список установленных версий:


asdf list nodejs


Также можно создать файл(ы) .tool-versions для автоматического переключения Node при смене директорий (см. основную заметку про asdf).
