1. Перейти в директорию /tmp


cd /tmp


2. Скачать архив LibreOffice:

2.1. Зайти на http://download.documentfoundation.org/libreoffice/stable/

2.2. Выбрать нужную версию.

2.3. Скопировать путь к файлу /rpm/x86_64/, например:


http://download.documentfoundation.org/libreoffice/stable/6.0.3/rpm/x86_64/LibreOffice_6.0.3_Linux_x86-64_rpm.tar.gz


2.4. Скачать файл, например, через wget:


---------------------------- On 32-bit Systems ----------------------------

# wget http://download.documentfoundation.org/libreoffice/stable/6.0.0/rpm/x86/LibreOffice_6.0.0_Linux_x86_rpm.tar.gz

---------------------------- On 64-bit Systems ----------------------------

# wget http://download.documentfoundation.org/libreoffice/stable/6.0.0/rpm/x86_64/LibreOffice_6.0.0_Linux_x86-64_rpm.tar.gz


3. Удалить предыдущие версии LibreOffice:


sudo yum remove openoffice* libreoffice*


4. Распаковать архив:


tar -xvf LibreOffice_*.tar.gz


5. Перейти в директорию:


cd /tmp/LibreOffice_*/RPMS/

dnf install *.rpm


6. Запустить программу обычным способом (через меню).



7. Установить языковой пакет для проверки орфографии.

7.1. Скачать из той же папки, что и в пункте 2.3, файл LibreOffice_6.0.2_Linux_x86-64_rpm_langpack_ru.tar.gz

7.2. Распаковать:

tar -xvf LibreOffice_*_rpm_langpack_ru.tar.gz

7.3. Перейти в распакованную директорию и установить все пакеты в текущей директории:

cd LibreOffice_*_rpm_langpack_ru/RPMS/

dnf install *.rpm
