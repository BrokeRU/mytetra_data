Загрузить расширение:


%load_ext autoreload


Выбрать режим для перезагрузки:


%autoreload -- перезагрузить все модули (кроме исключенных через %aimport) сейчас

%autoreload 0 -- отключить автоматическую перезагрузку модулей

%autoreload 1 -- перечитывать модули, импортированные через %aimport перед выполнением каждой команды

%autoreload 2 -- перечитывать все модули (кроме исключенных через %aimport) перед выполнением каждой команды
