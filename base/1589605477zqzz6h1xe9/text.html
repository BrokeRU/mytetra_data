1. Запустить cmd с правами администратора;

2. Выполнить команду:


choco install teamviewer


Нужно иметь ввиду, что разработчик TeamViewer имеет обыкновение публиковать несколько разных сборок программы под одной и той же версией. Похоже, что Chocolatey не поддерживает несколько различных хешей для одной версии программы. Из-за этого только одна сборка в рамках версии может считаться валидной. Таким образом, первая опубликованная сборка в рамках версии будет устанавливаться без проблем, а для последующих будет ошибка проверки контрольных сумм.
