1. Добавить PPA-репозиторий:


sudo add-apt-repository ppa:tista/adapta


2. Установить тему и фоны Adapta:


sudo apt install adapta-gtk-theme adapta-backgrounds


3. Через Gnome Tweaks выбрать тему Adapta и настроить шрифты (Roboto и Noto) в соответствии с данными отсюда:


https://github.com/adapta-project/adapta-gtk-theme
