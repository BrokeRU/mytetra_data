SchemaSpy:

1. Скачать SchemaSpy: http://schemaspy.sourceforge.net/

2. Скачать JDBS Driver: http://jdbc.postgresql.org/download.html

3. Установить graphviz:


sudo apt install graphviz


4. Выполнить команду:


java -jar schemaSpy_5.0.0.jar -t pgsql -db database_name -host myhost -u username -p password -o ./schemaspy -dp postgresql-9.3-1100.jdbc3.jar -s public -noads


В моем случае эта команда выглядела так:


java -jar schemaSpy_5.0.0.jar -t pgsql -db map -host localhost -u postgres -p postgres -o ./schemaspy -dp postgresql-42.2.2.jar -s public -noads


5. Чтобы преобразовать схемы в SVG, нужно выполнить следующую команду на .dot-файл:


dot -Tsvg filename.dot -o outfile.svg
