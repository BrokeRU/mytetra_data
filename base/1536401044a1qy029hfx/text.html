1. Установить:


sudo dnf install guake


2. Т.к. в Fedora используется экранный менеджер Wayland, который ещё плохо поддерживается Guake, то могут возникнуть проблемы с открытием окна Guake по комбинации клавиш. Нужно в настройках клавиатуры GNOME добавить это же сочетание клавиш с командой `guake -t`.
