Подключиться к локальному серверу MySQL из терминала (командной строки) в интерактивный режим:

$ mysql -u root -p

Enter Password: ******


Подключиться к удаленному серверу и выполнить команду:

$ mysql --host *HOST_IP* --user *USER* --password=*PASSWORD* -D *DATABASE_NAME* -e "DELETE FROM `wp_posts` WHERE `post_type` = "attachment" ;


Показать список БД:

SHOW DATABASES;


Выбрать БД:

USE wordpress;


Показать список пользователей:

SELECT User FROM mysql.user;


Сменить пароль пользователя:

SET PASSWORD FOR 'jeffrey'@'localhost' = PASSWORD('cleartext password');

FLUSH PRIVILEGES;


Показать список таблиц в выбранной БД:

SHOW TABLES;
