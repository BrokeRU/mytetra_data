Если нужно импортировать файлы из директории, которая не находится в стандартном месте для библиотек Python, то:


1. можно добавить путь в переменную PYTHONPATH программно:


import sys, os

myPath = os.path.dirname(os.path.abspath(__file__))

sys.path.insert(0, myPath + '/../')


2. или установить эту переменную вручную:


PYTHONPATH=../ python script.py
