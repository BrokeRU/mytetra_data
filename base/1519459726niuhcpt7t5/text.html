При использовании директивы v-model с элементами HTML-форм возможно добавление следующих модификаторов:


.lazy

===

По умолчанию v-model синхронизирует ввод с данными при каждом изменении. Можно сделать поле ввода "ленивым", тогда значение будет обновлено только после окончания ввода (переход в другое поле ввода):

<!-- синхронизируется после "change", а не "input" -->
<input v-model.lazy="msg" >


.number

=======

Значение, введенное в поле, автоматически будет конвертировано в число.

<input v-model.number="age" type="number">


.trim

=====

Обрезает пробельные символы в начале и конце строки.

<input v-model.trim="msg">
