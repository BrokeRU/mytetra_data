Чтобы пользователь в Ubuntu мог использовать команду sudo, необходимо, чтобы он состоял в группе sudo:


sudo adduser <username> sudo


или:


sudo usermod -a -G sudo <username>


Также, чтобы дать пользователю полные администраторские права, возможно, его понадобится добавить в следующие группы (желательно посмотреть, как сделано у других пользователей в файле /etc/passwd):

adm sudo lpadmin sambashare
