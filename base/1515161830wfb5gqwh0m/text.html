1. Установить набор скриптов для оптимизации NeteXt'73:

sudo add-apt-repository ppa:netext/netext73

sudo apt update

sudo apt install netext73

2. Запустить первый скрипт "The first installation";

3. Перезагрузить компьютер;

4. При необходимости переключить видеодрайвер с проприетарного nvidia на nouveau;

5. Желательно удалить PPA из списка репозиториев, т.к. могут возникнуть проблемы с некоторыми библиотеками.

В меню выключения (появляется по нажатию на кнопку питания) должны появиться пункты "Ждущий режим" и "Спящий режим" (гибернация). Пользоваться можно только "Ждущим режимом", потому что гибернация не работает (компьютер почему-то не может восстановить данные с диска).
