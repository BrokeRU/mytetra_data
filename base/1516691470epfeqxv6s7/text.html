Node.js -- программная платформа; интерпретатор языка JavaScript, позволяющий транслировать команды языка в машинный код и выполнять JavaScript не только в браузере, но и на любой машине, где установлен интерпретатор.


Создадим файл helloworld.js:

console.log("Hello World!");


Теперь можем выполнить:

node helloworld.js

И получим на вывод:

Hello World!


Создадим файл listdir.js:

const listFolder = '.';

const fs = require('fs');


fs.readdir(listFolder, (err, files) => {

files.forEach(file => {

console.log(file);

});

})


Выполним команду:

node listdir.js

В выводе получим список файлов в текущей директории.
