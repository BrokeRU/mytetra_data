1. Закрыть все процессы Chrome и выполнить команду:

chromium-browser --disable-web-security --user-data-dir


2. Для новых (после 48) версии Chrome требуется указать директорию в флаге --user-data-dir, которая в Linux расположена по пути:

~/.config/chromium, итого команда может принять вид:

chromium-browser --disable-web-security --user-data-dir=~/.config/chromium


bug: https://bugs.chromium.org/p/chromium/issues/detail?id=575690

doc: https://chromium.googlesource.com/chromium/src/+/master/docs/user_data_dir.md
