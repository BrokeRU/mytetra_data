1. Установить wget:

sudo dnf install -y wget


2. Скачать и установить TeamViewer:

wget https://download.teamviewer.com/download/linux/teamviewer.x86_64.rpm


3. Установить TeamViewer:

sudo dnf -y install teamviewer.x86_64.rpm


4. Запустить TeamViewer обычным способом (через меню приложений) или через командную строку:

teamviewer
