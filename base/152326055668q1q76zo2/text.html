1. Переключиться в режим суперпользователя:

su -

2. Установить wget:

dnf install -y wget

3. Импортировать публичный ключ TeamViewer:

rpm --import https://dl.tvcdn.de/download/linux/signature/TeamViewer2017.asc

4. Скачать и установить TeamViewer:

wget https://download.teamviewer.com/download/teamviewer.i686.rpm

dnf install -y teamviewer.i686.rpm

5. Запустить TeamViewer обычным способом (через меню приложений) или через командную строку:

teamviewer
