По умолчанию Compose берет префикс из названия папки, в которой лежит docker-compose.yml.


Указать собственный префикс можно через аргумент -p / --project-name (необходимо указывать для всех операций docker-compose), либо через переменную окружения COMPOSE_PROJECT_NAME.


$ docker-compose -p MY_PROJECT_NAME
