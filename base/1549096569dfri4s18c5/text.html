Чтобы просмотреть длинный результат иногда бывает полезно сменить пейджер на другой, например:


> \pager cat

PAGER set to cat.


Сменить обратно:


> \pager less

PAGER set to less.


Можно настроить опции less:


  • -X -- оставить содержимое файла в консоли после выхода, не очищать
  • -F -- выйти из less, если содержимое файла и так входит на экран целиком
  • -R -- отображать цвета ANSI в raw-формате
  • -S -- отключить перенос строк по границам слов, тогда появится горизонтальный скролл


# This is a popular option among pgcli users.

export LESS="-XFR"


# Some pgcli users like to disable line wrapping.

export LESS="-SRXF"
