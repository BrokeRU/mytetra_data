Допустим, по какой-то причине нам нужно указать в коммите автором другое имя и почту, не те, которые настроены в данные момент глобально или локально на уровне репозитория. Это можно сделать следующей командой:


git commit --author="John Doe <john@doe.org>"


Эта команда не изменяет настройки, так что следующие коммиты без указания автора будут использовать автора по умолчанию.
