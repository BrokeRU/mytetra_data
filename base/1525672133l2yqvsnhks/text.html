1. Скачать образ:


docker pull dpage/pgadmin4


2. Запустить контейнер (при необходимости поменять порт):


docker run -p 80:80 \
--name pgadmin4 \

-e "PGADMIN_DEFAULT_EMAIL=user@domain.com" \
-e "PGADMIN_DEFAULT_PASSWORD=SuperSecret" \
-d dpage/pgadmin4


3. При необходимости, подключить контейнер pgAdmin к другому контейнеру:


docker network connect map_default pgadmin4
