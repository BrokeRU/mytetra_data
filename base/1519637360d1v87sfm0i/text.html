Установить систему управления задачами и баг-трекер YouTrack от JetBrains. Бесплатная лицензия позволяет использовать систему неограниченное время, но ограничивает использование системы лишь десятью пользователями.


1. Создать директории для хранения данных:

mkdir -p /var/lib/youtrack

mkdir -p /var/log/youtrack

mkdir -p /etc/youtrack


2. Создать в системе пользователя youtrack и дать ему права:

sudo useradd -u 2000 youtrack

sudo chgrp -R youtrack /var/lib/youtrack/

sudo chmod -R g+rwx /var/lib/youtrack/

sudo chgrp -R youtrack /etc/youtrack/

sudo chmod -R g+rwx /etc/youtrack/


3. Создать Docker-контейнер с YouTrack:

docker create -t -i -p 8080:8080 -v /var/lib/youtrack:/var/lib/youtrack -v /var/log/youtrack:/var/log/youtrack -v /etc/youtrack:/usr/local/youtrack/conf -v /tmp:/tmp --name docker-youtrack --restart unless-stopped dzwicker/docker-youtrack


4. Запустить Docker-контейнер:

docker start docker-youtrack


5. YouTrack будет доступен по IP-адресу хостовой машины на порту 8080.
