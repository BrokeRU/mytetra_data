1. Создать файл ~/.pgpass с содержимым по следующему формату:


hostname:port:database:username:password


Можно использовать *, например:


localhost:*:*:postgres:123456

work-server.com:5432:market:supersecret:P@ssw0rd


2. Поставить на файл права 0600 или меньше:


chmod 0600 ~/.pgpass


3. Подключиться без пароля:


psql -h localhost -u postgres
